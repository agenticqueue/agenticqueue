# Security Policy

## Reporting a Vulnerability

Please report security issues privately. Do not open a public GitHub issue for a suspected vulnerability.

Preferred path:

1. Open a private GitHub security advisory in this repository.
2. Include reproduction steps, affected versions or branches, impact, and any mitigations you have verified.

Maintainer contact:

- Mario Watson
- Public GitHub profile: https://github.com/mario-watson

If GitHub security advisories are unavailable, use the maintainer profile above to establish a private contact path before sharing sensitive details.

## Disclosure Process

- We will acknowledge receipt as soon as practical.
- We will validate the report, assess impact, and work on a fix.
- Please avoid public disclosure until a fix or mitigation is available.

## Scope

This policy applies to this repository and the official AgenticQueue project-maintained infrastructure.
